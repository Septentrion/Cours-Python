# La réflexivité des objets

La réflexivité est la capacité des objets à s'observer eux-mêmes.

De ce point de vue, Python a des outils très riches :

Propriété | type | Signification
--------- | ---- | --------
__dict__ | dictionnaire |
__class__ | string |
__bases__ | tuple |
__name__ | string |
__qualname__ | string |
__mro__ | tuple
__subclasses__ | list
